1. 概述

SparkLearn WebUI支持多个AI服务商，您可以根据需要选择合适的服务商和模型。配置完成后，系统会自动保存到本地存储和后端配置文件中。

重要提示：科大讯飞星火是必配服务商，其他服务商为可选配置。

2. 科大讯飞星火（必配）

必配服务商

科大讯飞星火认知大模型是系统的核心服务商，提供强大的中文AI能力。您需要配置以下信息：

配置项：

spark_api_key

星火大模型的API密钥

获取地址：https://xinghuo.xfyun.cn/

APPID

科大讯飞文字识别APPID

获取地址：https://www.xfyun.cn/

APISecret

科大讯飞文字识别APISecret

APIKEY

科大讯飞文字识别APIKEY

可用模型：

4.0Ultra generalv3.5 max-32k generalv3 pro-128k lite

3. 智谱AI

智谱AI提供GLM系列模型，支持多种规格和性能等级。默认推荐使用GLM-4.5系列模型。

配置项：

glm_api_key

智谱AI的API密钥

获取地址：https://open.bigmodel.cn

可用模型：

glm-4.5 glm-4.5-air glm-4.5-x glm-4.5-airx glm-4.5-flash glm-4-plus glm-4-air-250414 glm-4-airx glm-4-flashx glm-4-flashx-250414 glm-z1-air glm-z1-airx glm-z1-flash glm-z1-flashx

4. SiliconFlow

SiliconFlow提供多种开源模型，支持本地部署和云端调用。

配置项：

silicon_api_key

SiliconFlow的API密钥

获取地址：https://siliconflow.cn/

可用模型：

deepseek-ai/DeepSeek-V3 zai-org/GLM-4.5 zai-org/GLM-4.5-Air moonshortai/Kimi-K2-Instruct Qwen/Qwen3-235B-A22B-Instruct-2507

5. OpenAI

OpenAI提供GPT系列模型，支持多语言处理，性能优秀。

配置项：

openai_api_key

OpenAI的API密钥

获取地址：https://platform.openai.com/account/api-keys

可用模型：

gpt-5 gpt-5-mini gpt-5-nano gpt-4.1

6. 使用说明

配置步骤：

  1. 访问API配置页面
  2. 选择服务商（点击对应的服务商卡片）
  3. 选择模型（从下拉菜单选择）
  4. 输入对应的API密钥
  5. 点击"保存配置"按钮

配置验证：

  • 保存成功后，页面会显示"配置已成功保存！"提示
  • 配置会自动保存到浏览器本地存储
  • 配置会同步到后端配置文件中
  • 重启应用后配置会自动恢复

注意事项：

  • • 请妥善保管您的API密钥，不要泄露给他人
  • • 科大讯飞星火是必配服务商，其他为可选
  • • 不同服务商的API密钥获取方式不同，请参考对应链接
  • • 配置变更后需要重新启动相关服务才能生效

7. 常见问题

Q: 配置保存失败怎么办？

A: 请检查网络连接和后端服务是否正常运行。如果问题持续，可以尝试刷新页面重新配置。

Q: API密钥无效怎么办？

A: 请检查API密钥是否正确，是否已激活，以及是否有足够的调用额度。

Q: 如何切换不同的服务商？

A: 在模型配置部分，点击不同的服务商卡片即可切换。切换后会自动显示对应的模型选项。

Q: 配置会丢失吗？

A: 配置会保存在浏览器本地存储中，除非清除浏览器数据，否则不会丢失。同时也会同步到后端配置文件。
